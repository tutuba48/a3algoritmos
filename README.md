# A3 - Algoritmos e Programação

## Contexto

Este projeto tem como objetivo desenvolver um **sistema de controle de estoque para uma pizzaria**.
O sistema permitirá o gerenciamento dos ingredientes e insumos utilizados na produção das pizzas,
facilitando o controle de entradas e saídas de produtos, evitando desperdícios e garantindo que
os itens necessários estejam sempre disponíveis para a operação do estabelecimento.

O sistema será desenvolvido em linguagem **Java**, utilizando conceitos de algoritmos e programação
estudados ao longo da disciplina.

## Tecnologias

- Java 21
- Maven 3

## Como Executar

1. Clone o repositório:
```
git clone https://github.com/tutuba48/a3algoritmos.git
```

2. Acesse a pasta do projeto:
```
cd a3algoritmos
```

3. Compile e execute com Maven:
```
mvn compile
mvn exec:java
```
